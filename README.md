# 🚗 3D Car Simulation in a City Environment

## 📌 Project Overview
This is a mid-term project that simulates car movement in a 3D city environment.  
The system behaves like a simple driving video game, allowing users to control a car and explore a virtual urban scene with adjustable camera perspectives.

---

## 🎮 Features
- 🚙 Real-time car movement (forward, backward, turning)
- 🏙️ 3D city environment rendering
- 🎥 Adjustable camera/viewing angles (follow camera / free view)
- 🎮 Keyboard controls for interactive driving experience
- 💡 Basic physics-based movement simulation

---

## 🛠️ Technologies Used
- C++
- OpenGL

---

## 🎯 Project Purpose
- Practice 3D graphics programming using OpenGL  
- Understand real-time rendering and transformation in 3D space  
- Implement basic game-like interaction and camera systems  
- Simulate vehicle movement in a virtual environment
